# QuickNunua

QuickNunua is a multi-category e-commerce platform built with **Django**. It allows users to browse, search, and purchase products ranging from furniture and kitchenware to electronics, clothing, and beauty products. The project demonstrates the integration of a modern backend with scalable architecture and a clean, modular design.

## Features

- User registration and authentication
- Product catalog with categories and search functionality
- Shopping cart and checkout system
- Admin dashboard for managing products, orders, and users
- Integration with PostgreSQL for data storage
- Designed for scalability with modular Django apps:
  - `accounts` – user management
  - `pages` – informational pages
  - `store` – product catalog and transactions
  - `theme` – templates and static assets

## Tech Stack

- Python 3.11+
- Django 4.x
- PostgreSQL
- HTML, CSS, JavaScript
- Railway deployment ready
- GitHub for version control
